### Repo for quick and vanilla D3.js prototyping 

#### Run static html server
```bash
python -m SimpleHTTPServer 8000
```

OR

```bash
http-server d3 -p 8000
```

### Open in browser
 
* http://localhost:8000/StackedBarChartSource.html

### Kill Server
```bash
ps aux | grep 8000
kill -9 PROCESS_NUMBER_HERE
```
